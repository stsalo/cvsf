Quick overviews:

cv-project: A computer vision project on the OSU Thermal pedestrian database. The project focuses on detection of pedestrians based on thermal images, and experiments with one- and two-stage detectors based on FRCNN and RetinaNet backbones.

sf-project: A sensor fusion project based on fusing depth and rgb image data. The goal was to fuse the two to fill the caveats of both data types for a more efficient method of image class detection. For this project I used a MobileNet-based approach, and tested a dual-stream model (with fused data), and compared it to single-stream models of standalone rgb and depth data.
